Project Name: WhatsOrder

One-line description:
WhatsOrder is a lightweight WhatsApp ordering and order management system for small restaurants in the UAE.

Problem:
Small restaurants receive many orders through WhatsApp, but the orders are unstructured and hard to track.

Solution:
WhatsOrder provides a digital menu, cart, checkout flow, structured WhatsApp order message, and restaurant admin dashboard.

Core features:
- Restaurant menu page
- Add to cart
- Checkout with customer details
- Click-to-WhatsApp order
- Order dashboard
- Menu management
- Customer database
- Basic analytics

Tech stack:
- Next.js
- TypeScript
- Tailwind CSS
- Supabase
